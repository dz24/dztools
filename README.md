# dztools
Work in progress python CLI for personal usage.

## Installation

1. Clone the repository
2. (Optional) Create a virtual environment
3. Install with:
   ```bash
   python -m pip install -e .
   ```
Or without cloning,

   ```bash
   pip install git+https://github.com/dz24/dztools.git
   ```

## Usage
```bash
dz -h
```
